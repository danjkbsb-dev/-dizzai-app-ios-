# 🚀 DizzAí! iOS

**Saiba como está o lugar AGORA!**

Um app inovador de informação em tempo real sobre movimentação e ambiente dos locais no Distrito Federal. Tipo um Waze para ambientes sociais.

## ✨ Funcionalidades

- 🗺️ **Mapa Interativo** - Visualize todos os lugares do DF em tempo real
- 📊 **Status em Tempo Real** - Veja lotação, vibe, preço, música e atendimento
- 📢 **Reports da Comunidade** - Usuários informam como está cada lugar
- 📰 **Feed de Atividades** - Acompanhe o que está acontecendo agora
- 🚨 **Alertas Inteligentes** - Receba notificações de lugares que você segue
- ❤️ **Favoritos** - Salve seus lugares preferidos
- 📝 **Histórico Pessoal** - Veja onde você já foi
- 🏆 **Ranking** - Descubra os melhores lugares
- 💬 **Chat por Lugar** - Converse com outros usuários no local
- 📱 **Compartilhamento Social** - Compartilhe com amigos

## 🛠️ Stack Técnico

- **Framework:** Expo + React Native
- **Linguagem:** TypeScript
- **Maps:** React Native Maps
- **Build:** GitHub Actions
- **Deployment:** Expo Cloud → App Store

## 📱 Como Usar

### Instalação Local

```bash
# Clonar repositório
git clone https://github.com/seu-usuario/dizzai-app-ios.git
cd dizzai-app-ios

# Instalar dependências
npm install

# Rodar em desenvolvimento
npm start

# Abrir no iPhone (com Expo Go)
# Escaneie o QR code com seu iPhone
```

### Rodar no Simulador

```bash
# iOS (requer Mac com Xcode)
npm run ios

# Android
npm run android

# Web
npm run web
```
